Part 6: Button Groups

Analysis of Breakpoints

Which row breaks first? The Tree Names row (using col-md) breaks first.

Explanation: In Bootstrap's grid system, md (medium) triggers the vertical stack at 768px, whereas sm (small) waits until the screen is narrower than 576px. Because 768 is larger than 576, the tree names will collapse into a vertical stack while the letters are still sitting side-by-side in a horizontal row.
